# ADR-011: Model Gateway Policy

Status: Proposed for R0 Gate

Source: `docs/package/codex/01_Adopted_Decisions_ADR.md` ADR-011. Trace: DEC-11, DEC-17, DEC-18.

## Context

AI features can expose privileged or sensitive matter material unless governed by permission, ethical-wall, and aiAllowed checks. External model calls add confidentiality and cost risks.

## Decision

Do not implement AI features before R6. R6 starts with local Gemma only, behind a model gateway and governance gate. External model SDKs and calls are prohibited until a later approved gate allows them.

## Consequences

`packages/ai` remains interface-only before R6. R2 may add schema columns such as `ai_allowed` and `ai_policy_id` with deny-by-default semantics, but no AI behavior. MVP readiness is the R6 AI Governance Gate, not a pre-governance feature shortcut.

## Review Triggers

Revisit at the R6 AI Governance Gate, after shadow-pilot evidence, or when a legal/security approval explicitly changes external model policy.
