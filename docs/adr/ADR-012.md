# ADR-012: Evaluation Dataset Strategy

Status: Proposed for R0 Gate

Source: `docs/package/codex/01_Adopted_Decisions_ADR.md` ADR-012. Trace: DEC-16, DEC-18.

## Context

Search, extraction, and later AI governance require repeatable evaluation data, but live confidential matter data cannot be used casually.

## Decision

Start with 20 to 50 de-identified closed-matter contracts for the R3 evaluation baseline. Expand toward an approximately 1,000-document subset for the R6 governance gate only through approved anonymization and handling controls.

## Consequences

Evaluation evidence is small but auditable in early releases. Dataset provenance, anonymization, and permission assumptions must be tracked before results can influence release gates.

## Review Triggers

Revisit during R3 evaluation setup, R6 governance-gate preparation, and after shadow-pilot findings.
