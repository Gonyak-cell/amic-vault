# ADR-001: Deployment Model

Status: Proposed for R0 Gate

Source: `docs/package/codex/01_Adopted_Decisions_ADR.md` ADR-001. Trace: DEC-01.

## Context

AMIC Vault handles law-firm confidential material and must preserve data residency, private network boundaries, and future on-prem portability without binding application code to one cloud vendor.

## Decision

Deploy first to a domestic-region private cloud configuration and package every component as containers. Development uses `infra/docker-compose.dev.yml`. Cloud-provider-specific SDKs are not allowed in application code unless an approved ADR update explicitly permits them.

## Consequences

The same container images must work across development, staging, production, and future on-prem environments. Vendor-managed conveniences are intentionally limited; storage, queueing, and search choices must preserve portability.

## Review Triggers

Revisit when an on-prem customer requirement is signed, when R13 enterprise hardening starts, or when domestic data-residency regulation changes.
