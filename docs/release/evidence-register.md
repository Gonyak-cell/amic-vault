- `EV-RC-*`: release candidate evidence.

- `EV-DESKTOP-*`: desktop/PWA evidence refs.
- `EV-OUTLOOK-*`: Outlook add-in planning and deployment evidence refs.
- `EV-UAT-*`: UAT case refs.
- `EV-PILOT-*`: pilot entry evidence.
- `EV-PROD-*`: production gate and release evidence.
- `EV-LAI-PROD-*`: Local AI production technical readiness, governance, and
