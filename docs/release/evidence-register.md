## Evidence ID Rules

- `EV-RC-*`: release candidate evidence.
- `EV-STAGE-*`: staging opening evidence.
- `EV-SMOKE-*`: smoke automation output refs.
- `EV-DESKTOP-*`: desktop/PWA/native desktop release evidence refs.
- `EV-OUTLOOK-*`: Outlook add-in planning and deployment evidence refs.
- `EV-ONEDRIVE-*`: OneDrive migration planning and post-launch evidence refs.
- `EV-UAT-*`: UAT case refs.
- `EV-PILOT-*`: pilot entry evidence.
- `EV-PROD-*`: production gate and release evidence.
- `EV-LAI-PROD-*`: Local AI production technical readiness, governance, and
  enablement planning evidence.

Evidence records may include status, owner role, date, check IDs, and non-secret
external references only.
